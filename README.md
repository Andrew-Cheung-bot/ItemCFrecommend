# Introduction
- 使用Spring Boot框架与MybatisPlus来搭建后台服务
- 使用druid第三方数据源，提供数据库的监控
- 使用Redis+JWT的方式管理token
- 前端使用Vue2 + ElementUI布局搭建

# 项目技术结构
- maven
- SprinBoot
- MybatisPlus
- Shrio校验权限
- Reis，MySQL
- Nginx做动静分离
- Vue2 + ElementUI开发前端页面
